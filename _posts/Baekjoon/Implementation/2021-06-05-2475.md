## 접근하기

정수가 주어진다. 이 정수를 모두 제곱한 후 더해서 10의 나머지를 구한다.

<br>

## 내 풀이

```js
const input = require('fs')
  .readFileSync('src/baekjoon/implement/input', 'utf8')
  .trim()
  .split(' ')
  .map((el) => Number(el));

let sum = 0;
for (const num of input) {
  sum += Math.pow(num, 2);
}

console.log(sum % 10);
```

1. for문을 이용해서 제곱 후 `sum`에 더하고 10의 나머지를 구하여 출력한다.
