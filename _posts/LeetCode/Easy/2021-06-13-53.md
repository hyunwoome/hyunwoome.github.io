## 접근하기

숫자형 배열 `nums`가 주어진다. `nums`의 인접한 숫자들을 더해 가장 큰 수를 반환한다.

<br>

## 참고 코드

```js
var maxSubArray = function (nums) {
  let omax = nums[0];
  let cmax = nums[0];
  for (let i = 1; i < nums.length; i++) {
    cmax = Math.max(nums[i], nums[i] + cmax);
    omax = Math.max(cmax, omax);
  }
  return omax;
};
```
