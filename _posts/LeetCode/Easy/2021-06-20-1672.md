## 문제 설명

숫자형 2차원 배열 accounts가 주어진다.
이 배열의 요소인 각 배열들의 합을 구하고, 가장 큰 수를 반환하는 문제이다.

<br>

## 내 코드

```js
var maximumWealth = function (accounts) {
  let answer = 0;
  accounts.forEach((array) => {
    let sum = array.reduce((acc, cur) => acc + cur);
    answer = Math.max(sum, answer);
  });
  return answer;
};
```
