## 접근하기

정수형 배열이 주어진다.

이 배열을 정렬해서 반환하는데, 원소가 짝수인 정수를 배열의 앞에 위치시키고, 홀수 정수를 뒤에 위치시켜 반환한다.

<br>

## 내 코드

```js
var sortArrayByParity = function (nums) {
  let even = [];
  let odd = [];
  for (let x of nums) {
    if (x % 2) odd.push(x);
    else even.push(x);
  }
  let answer = [...even, ...odd];
  return answer;
};
```

1. 주어진 정수를 순회하면서 조건에 맞게 배열에 담아 마지막에 합쳐 반환한다.
