## 접근하기

주어진 문자열 배열에서 `'Kim'`이라는 문자열을 찾고, 해당 문자열의 인덱스를 반환한다.

<br>

## 내 코드

```js
function solution(seoul) {
  for (let i = 0; i < seoul.length; i++) {
    if (seoul[i] === 'Kim') return `김서방은 ${i}에 있다`;
  }
}
```

1. `for`문을 이용해 배열을 순회하고, `'Kim'`을 찾아 반환한다.
