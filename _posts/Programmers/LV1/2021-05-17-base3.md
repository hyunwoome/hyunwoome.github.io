## 접근하기

주어진 수를 3진수으로 변환 후 뒤집는다. 그리고 다시 10진수로 변환 후 반환한다.

<br>

## 내 코드

```js
function solution(n) {
  const base3 = n.toString(3);
  const toArr = base3.split('').reverse();
  const toStr = toArr.join('');
  return parseInt(toStr, 3);
}
```
