## 접근하기

주어진 숫자 x만큼 n번 반복하는 문제이다.

<br>

## 내 코드

```js
function solution(x, n) {
  var answer = [];
  for (let i = 1; i <= n; i++) {
    answer.push(x * i);
  }
  return answer;
}
```
